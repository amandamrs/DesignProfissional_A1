# Avaliação A1 Design Profissional 
Repositório dedicado à Avaliação A1 de Design Profissional. Contém entregáveis práticos que refletem meu posicionamento profissional, organização digital estruturada e conexão efetiva com o cenário tecnológico atual.

| **1. Certificado de Participação em Evento** | Participação em evento acadêmico ou profissional (simpósio, palestra, hackathon etc.) | Certificado digital ou comprovante de inscrição | - | **✅ Concluído** |
| **2. Currículo Lattes ou em LaTeX** | Elaboração de currículo acadêmico ou técnico com foco em TI | Link do currículo Lattes ou PDF gerado via LaTeX | 1,15 | **✅ Concluído** |
| **3. Portfólio no GitHub** | Repositório profissional com README e projetos relevantes | Link do GitHub com README estruturado e pelo menos 1 projeto | 1,15 | **✅ Concluído** |
